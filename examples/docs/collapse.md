# collapse 折叠面板

## 组件名

折叠面板：`si-collapse`

## 说明

通过点击折叠面板的折叠按钮，面板内容区块能够切换显示和隐藏。

## 属性

| 属性    | 描述         | 类型   | 默认值                | 是否必须 |
| ------- | ------------ | ------ | --------------------- | -------- |
| title   | 折叠面板标题 | String | -                     | yes      |
| bgColor | 背景色       | String | light-green-lighten-3 | no       |
| fgColor | 前景色       | String | black                 | no       |
