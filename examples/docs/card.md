# card 卡片

## 组件名

卡片：`si-card`

## 说明

卡片是带有内边距和背景色的容器组件，常用于包裹一个内容区块。

## 属性

### si-card

| 属性    | 描述                                                     | 类型   | 默认值                | 是否必须 |
| ------- | -------------------------------------------------------- | ------ | --------------------- | -------- |
| bgColor | 卡片背景色                                               | String | light-green-lighten-3 | no       |
| shadow  | 是否带有一个浮起来的阴影，取值`always`、`hover`、`never` | String | never                 | no       |
| style   | 额外CSS，用于覆盖默认样式                                | Object | -                     | no       |
